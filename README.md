simple fashion e-commerce front-end website built with HTML, CSS, Bootstrap, and JavaScript. It includes a responsive homepage, shop page, product detail page, cart page, contact page, and basic cart interaction features.

## Features

- Responsive fashion store homepage with featured products and promotional sections.
- Shop page with multiple product listings, prices, and size details.
- Product detail page with image preview, size selection, and add-to-cart option.
- Cart page with quantity updates, subtotal calculation, item removal, and coupon support.
- Contact page with business details and support information.[

## Tech Stack

- HTML5 
- CSS3
- Bootstrap 5 
- JavaScript 
- Font Awesome

## Project Structure

```text
.
├── index.html
├── shop.html
├── product.html
├── card.html
├── contact.html
├── blog.html
└── script.js
```

The project is organized as a static front-end website with separate pages for store browsing, product details, cart flow, blog content, and contact information.

## Pages

- `index.html` – Homepage with featured sections and product highlights.
- `shop.html` – Product listing page for browsing store items
- `product.html` – Product detail page for a selected item.
- `card.html` – Shopping cart page with total calculation and coupon logic.
- `contact.html` – Contact page with address, phone, email, and store hours.
- `blog.html` – Blog or promotional content page.

## Getting Started

1. Clone or download the repository.
2. Keep all HTML files, JavaScript, and image assets in the same folder structure.
3. Open `index.html` in your browser to run the project locally.
4. Upload the project to GitHub Pages or any static hosting platform to publish it online.

## Notes

This project is a front-end template and does not include backend features such as authentication, payments, or database integration.
The shopping cart behavior is handled with client-side JavaScript.
